WPF app "TheaterActorEmployment"

АИС «Занятость актеров театра» — учебное WPF-приложение на языке C#, предназначенное для учета актеров, спектаклей, ролей и персональных контрактов.

## Назначение проекта

Система помогает коммерческому директору театра хранить и обрабатывать информацию об актерах, спектаклях, ролях, бюджете постановок и заключенных контрактах.

## Основные функции

- учет актеров;
- учет спектаклей;
- назначение актеров на роли;
- оформление персональных контрактов;
- поиск информации;
- просмотр отчетных данных.

## Технологии

- C#
- WPF
- .NET
